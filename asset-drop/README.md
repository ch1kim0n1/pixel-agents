# Asset Drop Zones

Put newly downloaded packs here first (unmodified):

- `asset-drop/jik-a-4/raw/`
- `asset-drop/furniture-pack/raw/`
- `asset-drop/licenses/` (license text, attribution notes, URLs)

Runtime/processed targets used by Pixel Agents:

- Characters: `webview-ui/public/assets/characters/`
- Furniture pack: `webview-ui/public/assets/furniture/`
  - requires `furniture-catalog.json`

Do not manually edit files in `dist/`; those are build outputs.
